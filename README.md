# HRV-python-chart-tool
Script to draw charts from Polar HRV data
Requirements:
- Python
- Polar Night data (Nightly_recovery_xxx..xx.json)

Usage:
1. Run cli command - this will take the raw json data and put it into a new file
> ./parse.sh
   2. run
> py draw.py
